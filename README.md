# 🎓 Graduation Party Poster Design

## 📖 About the Project
This is a graduation party poster design created for celebrating the **Class of 2025**. The design highlights friendship, success, and memorable graduation moments through silhouettes of graduates celebrating at sunset.

The poster follows a clean and premium design style with a black and gold theme to create an elegant and professional look.
